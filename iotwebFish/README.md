# Smart Fish Feeder IoT

Dashboard IoT untuk sistem pemberian pakan ikan otomatis berbasis ESP32 dan Firebase Realtime Database.

## Fitur

* Monitoring berat pakan secara realtime
* Pengaturan jadwal pemberian pakan
* Kontrol manual servo
* Riwayat pemberian pakan
* Status stok pakan
* Firebase Realtime Database

## Teknologi

* HTML
* CSS
* JavaScript
* Firebase Realtime Database
* ESP32
* RTC DS3231
* HX711
* Servo Motor
